# 隐藏暂时不用的功能模块 — 设计规格

## 背景

需要隐藏一批暂时不用的 UI 模块，简化界面展示。采用 HTML 模板注释方式（`<!-- ... -->`），未来需要恢复时取消注释即可。

## 修改文件清单

### 文件 1：`src/components/PageNav.vue`（左侧导航栏）

隐藏以下 5 个元素：

| # | 元素 | 说明 |
|---|------|------|
| 1 | 个人头像区域 | 顶部整个头像 `<div class="py-4 px-3">` 块 |
| 2 | "直播"菜单项 | `<a>` 标签，activeTab === 'live' |
| 3 | "小工具"菜单项 | `<a>` 标签，activeTab === 'tool' |
| 4 | "工作流"菜单项 | `<a>` 标签，activeTab === 'workflow' |
| 5 | "设置"菜单项 | `<a>` 标签，activeTab === 'setting' |

**保留的菜单项：** 仪表盘、数字人、AI模型

### 文件 2：`src/pages/Home.vue`（仪表盘页面）

隐藏以下 5 个区域：

| # | 区域 | 说明 |
|---|------|------|
| 1 | 标题右侧"使用遇到问题？发帖求助"链接 | `<a href="...forum">` 整块 |
| 2 | 金刚区"智能直播"导航卡片 | 导航卡片第 4 张（live） |
| 3 | 数据统计整体 | `data-auto-test-id="home-statistics"` 整块，含语音合成/数字人/小工具/智能直播四个统计面板 |
| 4 | 模型市场 + 工单反馈 | `grid-cols-3` 卡片区域，含模型市场和工单反馈两张卡片 |
| 5 | 下方小工具区所有工具 | `data-auto-test-id="home-tool-apps"` 整块 |

**保留的仪表盘内容：**
- 标题行（去掉右侧求助链接）
- 导航卡片：语音合成、数字人合成、小工具（3 张）

## 实现方式

使用 HTML 模板注释 `<!-- ... -->` 包裹需要隐藏的代码块。

## 不改动的部分

- `src/router.ts` — 路由定义保留，隐藏菜单后用户无法通过 UI 访问，但路由仍存在
- `activeTab` 计算属性 — 保留现有逻辑，无副作用
- `onMounted` 中的数据加载 — 保留，不影响运行（数据不会显示但不报错）
- `usageData` 相关导入 — 保留，避免引入不必要的 import 清理

## 验证标准

1. 程序启动后左侧导航栏只显示 3 个菜单项：仪表盘、数字人、AI模型
2. 仪表盘页面只显示标题 + 3 张导航卡片（语音合成、数字人合成、小工具）
3. 被隐藏的功能模块不影响程序正常运行
4. 取消注释后可恢复所有功能
