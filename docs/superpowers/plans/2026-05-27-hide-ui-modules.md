# 隐藏暂时不用的功能模块 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 通过 HTML 模板注释隐藏暂时不用的 UI 模块，简化界面展示。

**架构：** 修改 2 个 Vue 组件的 template 部分，用 `<!-- ... -->` 注释掉指定的 UI 区块。不涉及逻辑层修改。

**技术栈：** Vue 3 SFC（单文件组件）

---

## 文件结构

| 文件 | 操作 | 职责 |
|------|------|------|
| `src/components/PageNav.vue` | 修改 template | 隐藏头像区域和 4 个菜单项 |
| `src/pages/Home.vue` | 修改 template | 隐藏求助链接、智能直播卡片、数据统计、模型市场+工单反馈、小工具区 |

---

### 任务 1：隐藏 PageNav.vue 中的元素

**文件：**
- 修改：`src/components/PageNav.vue`（template 部分）

- [ ] **步骤 1：注释掉个人头像区域**

将 template 中头像整块注释掉。原始代码范围：从 `<div class="py-4 px-3"` 到其闭合 `</div>`（包含 `<a-tooltip>` 和 `<div v-else>` 两个分支）。

用 `<!-- 隐藏：个人头像 -->` 和 `<!-- /隐藏：个人头像 -->` 包裹。

- [ ] **步骤 2：注释掉"直播"菜单项**

将 activeTab === 'live' 对应的 `<a>` 标签整块注释掉。包含"直播"文字和 `<icon-live-broadcast>` 图标。

用 `<!-- 隐藏：直播菜单 -->` 和 `<!-- /隐藏：直播菜单 -->` 包裹。

- [ ] **步骤 3：注释掉"小工具"菜单项**

将 activeTab === 'tool' 对应的 `<a>` 标签整块注释掉。包含"小工具"文字和 `<icon-tool>` 图标。

用 `<!-- 隐藏：小工具菜单 -->` 和 `<!-- /隐藏：小工具菜单 -->` 包裹。

- [ ] **步骤 4：注释掉"工作流"菜单项**

将 activeTab === 'workflow' 对应的 `<a>` 标签整块注释掉。包含"工作流"文字和 `<WorkflowIcon>` 组件。

用 `<!-- 隐藏：工作流菜单 -->` 和 `<!-- /隐藏：工作流菜单 -->` 包裹。

- [ ] **步骤 5：注释掉"设置"菜单项**

将 activeTab === 'setting' 对应的 `<a>` 标签整块注释掉。包含 `$t("common.setting")` 文字和 `<icon-settings>` 图标。

用 `<!-- 隐藏：设置菜单 -->` 和 `<!-- /隐藏：设置菜单 -->` 包裹。

- [ ] **步骤 6：Commit**

```bash
git add src/components/PageNav.vue
git commit -m "feat: 隐藏个人头像、直播、小工具、工作流、设置菜单"
```
